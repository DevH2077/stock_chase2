# 📱 휴대폰에서 사용하는 방법

주식 시세 PWA를 휴대폰에 설치하고 사용하는 방법입니다.

## 🚀 빠른 시작

### 1단계: 웹 서버에 업로드
PWA는 HTTPS 환경에서만 작동합니다. 다음 중 하나를 선택하세요:

**옵션 A: 무료 호스팅 서비스 사용**
- [Netlify](https://www.netlify.com/) - 무료 HTTPS 제공
- [Vercel](https://vercel.com/) - 무료 HTTPS 제공
- [GitHub Pages](https://pages.github.com/) - 무료 HTTPS 제공

**옵션 B: 로컬 서버 사용 (개발/테스트용)**
- 파일을 웹 서버에 업로드
- 또는 로컬 네트워크에서 접근 가능한 서버 사용

### 2단계: 모바일 브라우저에서 접속
- Chrome (Android) 또는 Safari (iOS)에서 사이트 주소 입력
- 사이트가 로드되면 설치 안내가 표시될 수 있습니다

---

## 📲 Android (안드로이드) 설치 방법

### Chrome 브라우저 사용

1. **Chrome 앱에서 사이트 열기**
   - Chrome 브라우저를 실행
   - 주소창에 사이트 주소 입력

2. **홈 화면에 추가**
   - 방법 1: 주소창 아래에 표시되는 "홈 화면에 추가" 배너 클릭
   - 방법 2: 메뉴(⋮) > "홈 화면에 추가" 선택
   - 방법 3: 주소창의 설치 아이콘(⬇) 클릭

3. **설치 확인**
   - "앱 추가" 또는 "설치" 버튼 클릭
   - 홈 화면에 앱 아이콘이 생성됩니다

4. **앱 실행**
   - 홈 화면의 앱 아이콘을 터치하여 실행
   - 독립적인 앱처럼 작동합니다

### Samsung Internet 사용

1. Samsung Internet 브라우저에서 사이트 열기
2. 메뉴(☰) > "홈 화면에 추가" 선택
3. 확인 버튼 클릭

---

## 🍎 iOS (아이폰/아이패드) 설치 방법

### Safari 브라우저 사용 (필수)

1. **Safari에서 사이트 열기**
   - ⚠️ Chrome이나 다른 브라우저에서는 설치할 수 없습니다
   - 반드시 Safari 앱을 사용해야 합니다

2. **공유 버튼 클릭**
   - 화면 하단의 공유 버튼(□↑) 클릭
   - 또는 화면 중앙을 아래로 스와이프

3. **홈 화면에 추가**
   - 공유 메뉴에서 "홈 화면에 추가" 선택
   - (스크롤해서 찾을 수 있습니다)

4. **이름 확인 및 추가**
   - 앱 이름 확인 (필요시 수정 가능)
   - 오른쪽 상단의 "추가" 버튼 클릭

5. **앱 실행**
   - 홈 화면에 앱 아이콘이 생성됩니다
   - 아이콘을 터치하여 실행

## ✨ 설치 후 사용법

### 기본 기능
- **주식 검색**: 상단 검색창에 심볼 입력 (예: AAPL, TSLA)
- **고정하기**: 핀 버튼(📍) 클릭 또는 드래그 앤 드롭
- **드래그**: 종목을 길게 누른 후 드래그하여 이동
- **새로고침**: 상단 오른쪽 새로고침 버튼(🔄) 클릭

### 터치 제스처
- **길게 누르기**: 드래그 시작 (200ms 이상)
- **드래그**: 고정 종목 ↔ 검색 기록 이동
- **탭**: 요약 카드 클릭 시 상세 정보로 전환

### 자동 업데이트
- **고정 종목**: 30초마다 자동 업데이트
- **모든 종목**: 5분마다 자동 업데이트

## 🔧 문제 해결

### 설치 옵션이 보이지 않아요
- **Android**: Chrome 최신 버전 사용 확인
- **iOS**: Safari 사용 확인 (다른 브라우저 불가)
- HTTPS 연결 확인 (http://는 작동하지 않음)

### 앱이 설치되지 않아요
- 브라우저 캐시 삭제 후 다시 시도
- 브라우저를 최신 버전으로 업데이트
- 인터넷 연결 확인

### 오프라인에서 작동하지 않아요
- Service Worker가 정상 등록되었는지 확인
- 브라우저 개발자 도구에서 확인 가능
- HTTPS 환경에서만 작동합니다

### 시간이 잘못 표시돼요
- 최신 버전으로 업데이트되었는지 확인
- 브라우저 새로고침 (강력 새로고침: Ctrl+Shift+R)

## 📋 추가 팁

### 홈 화면 바로가기
- 설치 후 홈 화면에서 바로 접근 가능
- 독립적인 앱처럼 작동 (브라우저 UI 없음)

### 데이터 저장
- 모든 데이터는 로컬 스토리지에 저장
- 브라우저 데이터 삭제 시 초기화됨

### 배터리 절약
- 고정 종목 자동 업데이트는 30초마다 실행
- 배경에서도 업데이트 가능 (브라우저 설정에 따라 다름)

## 💡 유용한 기능

1. **3칸 레이아웃**
   - 고정 종목: 항상 상단에 고정
   - 최신 검색: 가장 최근 검색한 종목
   - 검색 기록: 이전 검색한 종목들

2. **드래그 앤 드롭**
   - 종목을 길게 누른 후 드래그
   - 고정 종목 ↔ 검색 기록 이동 가능

3. **자동 새로고침**
   - 고정 종목은 30초마다 자동 업데이트
   - 실시간 시세 확인 가능
